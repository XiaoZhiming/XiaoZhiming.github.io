Trino源码学习 Query 生命周期

本文说明一个sql从提交到执行过程，串联起trino处理查询的整个流程

  1. sql(String) 创建 query （QueuedStatementResource Controller）
  2. query 创建 DispatchQuery （DispatchManager）
  3. 提交DispatchQuery
sequenceDiagram
note over QueuedStatementResource: sql创建query
client ->>  QueuedStatementResource : registerQuery
(sql)String->Query
QueuedStatementResource ->> Query:waitForDispatched
Query ->> DispatchManager:createQuery
note over DispatchManager:query创建DispatchQuery 
createQueryInternal
DispatchManager->>LocalDispatchQueryFactory:createDispatchQuery
Query,AST
LocalDispatchQueryFactory->>QueryStateMachine:begin
QueryStateMachine-->>LocalDispatchQueryFactory:QueryStateMachine
LocalDispatchQueryFactory->>QueryExecutionFactory:createQueryExecution(AST,QueryStateMachine)
note over QueryExecutionFactory:这里以SqlQueryExecution为例
QueryExecutionFactory->>SqlQueryExecution:new
SqlQueryExecution-->>QueryExecutionFactory:SqlQueryExecution
QueryExecutionFactory-->>LocalDispatchQueryFactory:QueryExecution
LocalDispatchQueryFactory->>LocalDispatchQuery:new(QueryStateMachine,QueryExecution)
LocalDispatchQuery-->>LocalDispatchQueryFactory:DispatchQuery
LocalDispatchQueryFactory-->>DispatchManager:DispatchQuery
Query->DispatchQuery
note over DispatchManager:开始执行
DispatchManager->>ResourceGroupManager:submit(LocalDispatchQuery)
ResourceGroupManager->>ResourceGroup:run(LocalDispatchQuery)
ResourceGroup->>QueryManager:createQuery(QueryExecution)
QueryManager->>SqlQueryExecution:start 这里以SqlQueryExecution为例
